# Tarea1-BFS-DFS
1. Lenguaje y versión

Lenguaje: Python

Versión utilizada: Python 3.14

El programa no requiere librerías externas.
Únicamente usa módulos estándar de Python (collections).

2. Cómo ejecutar el programa

Descargar el repositorio o descomprimir el archivo .zip.

Abrir una terminal en la carpeta del proyecto.

Ejecutar: python metro.py o py metro.py
Se imprimen automáticamente los resultados para:

Observatorio → Ciudad Azteca

Indios Verdes → Velódromo

UAM-I → El Rosario

Para cada caso y algoritmo (BFS y DFS) se muestra:

Ruta encontrada (lista ordenada de estaciones)

Longitud en aristas (saltos)

Número de nodos visitados
3. Notas importantes

El grafo está modelado como no dirigido y no ponderado.

Se utiliza lista de adyacencia para representar las conexiones.

BFS y DFS reconstruyen explícitamente la ruta usando diccionario de padres.
